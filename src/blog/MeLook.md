### 选项卡

```markdown
::: tabs

@tab 标题 1

<!-- tab 1 内容 -->


@tab 标题 2

<!-- tab 2 内容 -->

@tab:active 标题 3

<!-- tab 3 将会被默认激活 -->

<!-- tab 3 内容 -->

:::
```

::: tabs

@tab 标题 1

### 重要提醒

```markdown
::: important 自定义重要
:::

::: info 自定义信息
:::

::: note 自定义注释
:::

::: tip 自定义提示
:::

::: warning 自定义警告
:::

::: caution 自定义危险
:::
```
